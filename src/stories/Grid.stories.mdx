# Grid Component

Grid 컴포넌트는 CSS 그리드 레이아웃을 활용하여 자식 요소들의 레이아웃을 구성하는 유틸리티 컴포넌트입니다. 주요 특징은 다음과 같습니다:

- 원하는 HTML 태그로 렌더링할 수 있습니다 (as 속성).
- 그리드의 행(rows)과 열(columns)을 정의할 수 있습니다.
- 그리드 내의 간격(gap)을 설정할 수 있습니다.
- 추가적인 인라인 스타일(sx) 또는 CSS 클래스 이름(className)을 적용할 수 있습니다.
- 자식 요소(children)를 그리드 내에 배치할 수 있습니다.

이 컴포넌트는 CSS 그리드의 복잡성을 추상화하여 행, 열, 간격 등을 쉽게 설정할 수 있게 해 줍니다.
