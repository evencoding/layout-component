# Flex Component

Flex 컴포넌트는 CSS Flexbox를 이용하여 자식 요소들의 레이아웃을 쉽고 효과적으로 조절하는 유틸리티 컴포넌트입니다. 주요 기능은 다음과 같습니다:

- 원하는 HTML 태그를 사용하여 렌더링할 수 있습니다 (as 속성).
- Flexbox의 방향(direction), 정렬(justify, align), 그리고 간격(gap)을 커스터마이즈할 수 있습니다.
- 추가적인 인라인 스타일(sx) 또는 CSS 클래스 이름(className)을 적용할 수 있습니다.
- 임의의 자식 요소(children)를 포함할 수 있습니다.

이 컴포넌트는 Flexbox의 복잡성을 추상화하여 레이아웃을 빠르고 쉽게 조절할 수 있게 해줍니다.
