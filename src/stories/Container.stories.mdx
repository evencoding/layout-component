# Container Component

Container 컴포넌트는 레이아웃을 조절하는 유틸리티 컴포넌트입니다. 주요 기능은 다음과 같습니다:

- 원하는 HTML 태그를 사용하여 렌더링할 수 있습니다 (as 속성).
- 컨테이너의 최소 너비(minWidth)와 최대 너비(maxWidth)를 설정할 수 있습니다.
- 추가적인 인라인 스타일(sx) 또는 미리 지정된 CSS 클래스 이름(className)을 적용할 수 있습니다.
- 임의의 자식 요소(children)를 포함할 수 있습니다.

이 컴포넌트는 레이아웃을 편리하게 조절하면서도 유연성을 유지하도록 설계되었습니다.
